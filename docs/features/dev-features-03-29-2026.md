# What's New in Dev

## Program Templates & Assignment

### Template Library
Browse a library of pre-built therapy program templates (CBT for Depression, DBT Skills Training, ERP for OCD, and more). Use a template as a starting point for your own program, or assign it directly to a client.

### My Programs
Your own program library — programs you've built from scratch, customized from templates, or saved from a client's program. Everything in My Programs is fully editable and ready to assign to clients.

### Assign Programs to Clients
Assign any program to a client with per-client customization. During assignment you can see the full program tree (modules and parts) and uncheck anything that doesn't apply to that specific client. Each client gets their own independent copy — changes to one client's program never affect another client or the original.

### Re-assign Programs
Need to add more modules to a client's existing program? Assign the same program again and new modules get added after the existing content.

### Save as My Program
Found a great setup while working with a client? Save their program structure back to your library as a reusable template for future clients. Client progress and responses are not included — just the clean structure.

### Remove from Program
Remove a client from a program when treatment ends or changes direction. Their progress is preserved for your records but they no longer see the program content.

### Archive Programs
Archive programs you no longer need. They disappear from your program list but aren't deleted.

## Client Management

### Client Detail Breadcrumbs
The client detail page now shows breadcrumbs: **Clients > Client Name > Tab** for easy navigation.

### Dropped Clients Stay Visible
Clients removed from programs still appear in your client list so you can re-enroll them or track their history.

## Program Editor Improvements

### No More Draft/Published
Programs no longer need to be "published" before you can assign them to clients. Create a program and assign it whenever you're ready — the assignment step is the gate, not a status toggle.

### Cleaner Module View
Module subtitle field removed to reduce clutter. Modules now show title and description only.

### Cleaner Part Display
Removed redundant titles from Strategy Cards, Assessments, and Intake Forms that were showing the title twice.
