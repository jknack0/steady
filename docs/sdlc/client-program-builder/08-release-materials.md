# Client Program Builder — Release Materials

## Feature Summary

**What's new:** Clinicians can now create a custom treatment program from scratch directly for a specific client — no template required.

**Who it's for:** Clinicians who need a blank canvas to build individualized treatment programs that don't fit any existing template.

## Customer-Facing Description

### Create Custom Programs for Any Client

Not every client fits a template. Now you can build a completely custom program tailored to a specific client's needs — right from the Create Program dialog.

**How it works:**
1. Click "Create Program" on the Programs page
2. Choose "Create for Client"
3. Name your program and select (or add) a client
4. Start building — add modules, parts, and content at your own pace

Your client is enrolled immediately and can see content as you build it. The program lives in your Client Programs tab, keeping your My Programs clean.

**Love the program you built?** Promote it to a reusable template with one click so you can use it with future clients.

## Key Talking Points

- **No more workarounds.** Previously, clinicians had to create a throwaway template first, then assign it. Now it's one step.
- **Blank canvas.** Start with an empty program and one module. Add weekly modules, content, and homework as you go.
- **Client is enrolled on creation.** No separate assignment step — the client sees the program immediately.
- **Add new clients inline.** Don't have the client in the system yet? Add them right from the creation flow.
- **Promotes to template.** If a custom program works well, turn it into a reusable template for similar clients.
- **Stays organized.** Custom client programs appear in the Client Programs tab, not in My Programs.

## FAQ

**Q: Where does the custom program show up?**
A: In the Client Programs tab on the Programs page. It won't clutter your My Programs list.

**Q: Can I add more modules after creating the program?**
A: Yes. The program starts with one module ("Module 1") and you can add as many as you need from the program editor.

**Q: Does the client see the program right away?**
A: Yes. The client is enrolled with ACTIVE status on creation, so they'll see it in their mobile app immediately.

**Q: Can I turn this into a template later?**
A: Yes. Use the "Promote to Template" option in the program editor to save it as a reusable template in My Programs.

**Q: What if I don't see my client in the picker?**
A: Click "Add New Client" at the bottom of the dropdown to create a new client inline with just their name and email.

**Q: Is this different from assigning a template?**
A: Yes. Assigning a template deep-copies an existing program. "Create for Client" starts from a blank canvas — no source program needed.
