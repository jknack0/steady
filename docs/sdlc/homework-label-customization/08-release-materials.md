# Homework Label Customization — Release Materials

## What's New

Steady now gives therapists full control over how homework items are labeled throughout the platform. Instead of using generic system labels like "Action Item" or "Journal Prompt," clinicians can define their own terminology that reflects their therapeutic approach, practice branding, or the language their clients are most comfortable with. Custom labels appear everywhere participants interact with homework — including the mobile app — so the experience feels seamlessly personalized.

Customization works at two levels: set practice-wide defaults from Settings that apply across all your programs, or override the label on any individual homework item for more granular control. There is nothing to migrate and no setup required. Existing workflows remain unchanged until a clinician decides to customize.

## Key Benefits

- **Personalized client experience**: Participants see terminology that matches the language used in sessions, reducing confusion and reinforcing therapeutic concepts.
- **Practice branding**: Align every touchpoint with your practice's identity and methodology — no more generic labels that feel like someone else's platform.
- **Flexible without complexity**: Set defaults once in Settings and they apply everywhere, or fine-tune individual items when needed. Reset to system defaults at any time with one click.
- **Instant and retroactive**: Changing a default label immediately updates all homework items using it — no need to edit programs one by one.
- **Zero disruption**: Existing therapists see no changes until they choose to customize. The feature is entirely opt-in.

## How It Works

### Setting Practice-Wide Defaults

1. Navigate to **Settings** from the sidebar.
2. Find the **Homework Labels** section.
3. You will see all 11 homework item types listed with their current (system default) labels.
4. Click any label to edit it. Enter your preferred name (up to 50 characters).
5. Click **Save**. The new label immediately applies to all homework items of that type across all your programs.
6. To revert, click the **Reset** button next to any label to restore the system default.

### Overriding a Label on a Specific Item

1. Open any program and navigate to the module and part containing the homework item.
2. Click the homework item you want to customize.
3. In the item editor, locate the **Label** field. It will show the current label (either your clinician default or the system default).
4. Type a new label for this specific item (up to 50 characters).
5. Save the part. This item now displays the override label to participants, independent of your Settings defaults.
6. To revert to your default, clear the override field and save.

### How Label Resolution Works

The platform determines which label to display using this priority order:
- **Per-item override** (if set) takes highest priority
- **Clinician default** (if set in Settings) applies next
- **System default** is the fallback

## FAQ

**Q: Will changing a label affect homework that participants have already completed?**
A: The label updates retroactively in the interface, but completed homework entries and their data are unchanged. Participants will see the new label if they revisit past items.

**Q: Is there a limit to how many times I can change a label?**
A: No. You can update labels as often as you like. Changes take effect immediately.

**Q: Can I use different labels for different programs?**
A: Your Settings defaults apply across all programs. To use a different label for a specific item in a specific program, use the per-item override on that homework item. You do not need to change your global defaults.

**Q: What happens if I delete a custom label or reset to default?**
A: The system default label is restored immediately. Any per-item overrides remain in place — resetting a default only affects items that were relying on that default.

**Q: Do participants see the custom labels in the mobile app?**
A: Yes. Custom labels are fully client-facing and appear everywhere participants interact with homework, including the mobile app, task lists, and notifications.

**Q: What is the character limit for labels?**
A: Labels can be up to 50 characters. This accommodates descriptive names while keeping the interface clean across web and mobile.

**Q: Do I need to set up anything before using this feature?**
A: No. The feature is available immediately. Navigate to Settings to begin customizing labels whenever you are ready.
